# 图片转3D模型系统

基于TripoSR开源模型的本地图片转3D模型系统，能够从单张图片快速生成高质量的3D模型。

## 🚀 快速开始（Windows用户）

### 第一步：安装环境
1. 双击运行 `install.bat`
2. 等待自动安装完成（可能需要几分钟）

### 第二步：启动应用
1. 双击运行 `start.bat`
2. 浏览器会自动打开应用界面

### 第三步：生成3D模型
1. 在浏览器中上传一张图片
2. 点击"生成3D模型"按钮
3. 等待处理完成
4. 下载生成的3D模型文件

### 第四步：查看日志（可选）
- 运行日志保存在 `logs` 文件夹中
- 生成的模型保存在 `output` 文件夹中

## 📋 系统要求

- **操作系统**: Windows 10/11
- **显卡**: NVIDIA GPU（推荐RTX 2060 Super 12GB或更高）
- **内存**: 16GB或更多
- **Python**: 3.8或更高版本
- **网络**: 首次安装需要下载模型文件

## ⚙️ 参数说明

- **移除背景**: 自动移除图片背景，突出主体
- **前景比例**: 调整主体在图片中的占比（0.5-1.0）
- **网格分辨率**: 控制生成3D模型的精细度（64-512）
  - 256: 推荐设置，平衡质量和速度
  - 512: 高质量，但处理时间较长
  - 128: 快速处理，质量一般

## 📄 输出格式

- **OBJ格式**: 通用3D模型格式，可在Blender、3ds Max等软件中打开
- **GLB格式**: 紧凑的二进制格式，适合Web展示和移动应用

## ❓ 常见问题

### 安装失败
- 确保网络连接正常
- 以管理员身份运行 `install.bat`
- 检查Python是否正确安装

### 生成速度慢
- 降低网格分辨率到128或256
- 确保使用NVIDIA GPU而非CPU
- 关闭其他占用GPU的程序

### 模型质量不佳
- 使用高质量、清晰的图片
- 调整前景比例参数
- 确保图片主体清晰可见

## 🔧 技术支持
